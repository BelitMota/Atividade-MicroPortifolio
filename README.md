# Atividade Prática MicroPortifolio Roteiro

## Atividade Prática – Git e GitHub 
**Objetivo:**
Aplicar na prática os conceitos trabalhados em aula utilizando Git e GitHub, incluindo versionamento, branches, commits, issues e pull requests.

**Descrição da Atividade**
- Criar um miniprojeto simples (HTML/CSS/JavaScript ou outra tecnologia definida pelo professor).
- Criar um repositório no GitHub para armazenar o projeto.
- Realizar commits organizados durante o desenvolvimento do projeto.
- Criar e utilizar pelo menos 2 branches diferentes para desenvolvimento de funcionalidades ou alterações.
- Criar pelo menos 1 issue no repositório descrevendo uma tarefa ou melhoria.
- Resolver a issue e mencionar o número dela no commit realizado.
- Abrir uma Pull Request para realizar o merge das alterações feitas na branch criada.
- Adicionar o professor como colaborador ou enviar o link do repositório para acompanhamento e avaliação.
Subir em PDF o LINK para o professor Marcão, pelo Moodle.
- Criar um README explicando o objetivo do projeto e as funcionalidades implementadas.
